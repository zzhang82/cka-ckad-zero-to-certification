1. [Current CNCF curricula](https://github.com/cncf/curriculum) — open the current CKAD PDF and build the delta map from this authority.
2. [Dockerfile overview](https://docs.docker.com/build/concepts/dockerfile/) — image build inputs and layers.
3. [Images (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/containers/images/) — names, digests, pull policy, and registry behavior.
4. [Kubernetes workloads (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/workloads/) — choose the controller from requirements.
5. [Init containers](https://v1-35.docs.kubernetes.io/docs/concepts/workloads/pods/init-containers/) and [sidecar containers](https://v1-35.docs.kubernetes.io/docs/concepts/workloads/pods/sidecar-containers/) — lifecycle-aware multi-container patterns.
6. [Volumes (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/storage/volumes/) — shared, persistent, and ephemeral application storage.
