# Week 11 — Targeted Remediation

> Preview: this week is generated from actual evidence; its scenario picker is not yet implemented.

**Goal:** repair the lowest measured domain or prepare an eligible retake without restarting the whole curriculum.

**Before starting:** collect Week 6/10 mock evidence and any permitted exam feedback categories. Read [the focused resource rules](RESOURCES.md).

## Planned work

1. Classify misses by current blueprint objective and failure type.
2. Choose the smallest official reading and one original repeat scenario.
3. Repeat the same failure class with changed names, values, and topology.
4. Drill slow command retrieval separately from conceptual repair.
5. Stop only after two consecutive independent correct attempts.

## Gate

- The repeated failure class is zero across two attempts.
- The fix remains correct under a changed scenario.
- The evidence names the next exam or practice decision.

This week cannot be pre-declared `PASS`; it depends on the learner's measured weakness.
