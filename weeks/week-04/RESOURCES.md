# Week 4 Resources

## Read before practice

1. [Services (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/services-networking/service/) — selector, EndpointSlice, port, and access paths.
2. [NetworkPolicy (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/services-networking/network-policies/) — isolation and additive policy behavior.
3. [DNS for Services and Pods](https://v1-35.docs.kubernetes.io/docs/concepts/services-networking/dns-pod-service/) — search domains and names.
4. [Ingress (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/services-networking/ingress/) and [Gateway API](https://gateway-api.sigs.k8s.io/) — CKA networking surfaces; Gateway API changes independently, so lab versions must be pinned.
5. [StorageClasses](https://v1-35.docs.kubernetes.io/docs/concepts/storage/storage-classes/) and [dynamic provisioning](https://v1-35.docs.kubernetes.io/docs/concepts/storage/dynamic-provisioning/) — default class, binding, provisioner, and reclaim behavior.
6. [Persistent volumes](https://v1-35.docs.kubernetes.io/docs/concepts/storage/persistent-volumes/) — timed PV/PVC diagnosis reference.

Use the matching LFS258 networking and storage modules from your portal.
