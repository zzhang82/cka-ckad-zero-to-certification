# Week 8 Resources

## Read before practice

1. [Deployments (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/workloads/controllers/deployment/) — rollout, rollback, pause, and update behavior.
2. [Helm install](https://helm.sh/docs/helm/helm_install/), [upgrade](https://helm.sh/docs/helm/helm_upgrade/), and [rollback](https://helm.sh/docs/helm/helm_rollback/) — check the installed Helm major before following version-specific behavior.
3. [Kustomize with `kubectl`](https://v1-35.docs.kubernetes.io/docs/tasks/manage-kubernetes-objects/kustomization/) — bases, overlays, patches, and transforms.
4. [ConfigMaps](https://v1-35.docs.kubernetes.io/docs/concepts/configuration/configmap/) and [Secrets](https://v1-35.docs.kubernetes.io/docs/concepts/configuration/secret/) — application configuration and Secret safety.
5. [Resource management](https://v1-35.docs.kubernetes.io/docs/concepts/configuration/manage-resources-containers/), [security contexts](https://v1-35.docs.kubernetes.io/docs/tasks/configure-pod-container/security-context/), and [ServiceAccounts](https://v1-35.docs.kubernetes.io/docs/concepts/security/service-accounts/) — runtime constraints and identity.
6. [Custom resources](https://v1-35.docs.kubernetes.io/docs/concepts/extend-kubernetes/api-extension/custom-resources/), [Operator pattern](https://v1-35.docs.kubernetes.io/docs/concepts/extend-kubernetes/operator/), and [API access control](https://v1-35.docs.kubernetes.io/docs/concepts/security/controlling-access/) — application-facing extension and access boundaries.
