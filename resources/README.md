## Selection rules

- Linux Foundation and CNCF pages define exam policy and scope.
- Kubernetes v1.35 and official project documentation define the baseline verified on 2026-07-15; the live Linux Foundation page and CNCF curriculum replace it when the exam minor changes.
- LFS258 and LFS/LFD259 are paid course spines; this repository links and maps them but never copies them.
- Community exercises are optional, link-only, version-checked supplements—not exam truth.
- Anything claiming to contain real, recalled, or leaked exam questions is excluded.

Every exam-policy and version claim must be rechecked before a mock and again shortly before the real exam. See each week for the exact links.

Version-pinned `v1-35.docs.kubernetes.io` links are study references. During exam-mode rehearsals, navigate only from the domains and pages named by the live Linux Foundation allowlist; do not assume that a versioned subdomain is separately authorized.
