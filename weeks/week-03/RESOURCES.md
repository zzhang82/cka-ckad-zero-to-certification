# Week 3 Resources

## Read before practice

1. [Install kubeadm (v1.35)](https://v1-35.docs.kubernetes.io/docs/setup/production-environment/tools/kubeadm/install-kubeadm/) and [create a cluster](https://v1-35.docs.kubernetes.io/docs/setup/production-environment/tools/kubeadm/create-cluster-kubeadm/) — prerequisites, init, and join.
2. [Upgrade a kubeadm cluster](https://v1-35.docs.kubernetes.io/docs/tasks/administer-cluster/kubeadm/kubeadm-upgrade/) — ordering, drain, component versions, and verification.
3. [Highly available kubeadm clusters](https://v1-35.docs.kubernetes.io/docs/setup/production-environment/tools/kubeadm/high-availability/) — topology and failure boundaries.
4. [Helm documentation](https://helm.sh/docs/) — check the installed Helm major before following version-specific examples.
5. [Kustomize through `kubectl`](https://v1-35.docs.kubernetes.io/docs/tasks/manage-kubernetes-objects/kustomization/) — use tools expected on the task host.
6. [Custom resource definitions](https://v1-35.docs.kubernetes.io/docs/tasks/extend-kubernetes/custom-resources/custom-resource-definitions/) and [Operator pattern](https://v1-35.docs.kubernetes.io/docs/concepts/extend-kubernetes/operator/) — inspect and reason about extensions.
