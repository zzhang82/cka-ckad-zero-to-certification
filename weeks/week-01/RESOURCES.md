# Week 1 Resources

## Read before practice

1. [Kubernetes components (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/overview/components/) — map each component to the state it owns.
2. [Objects in Kubernetes (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/overview/working-with-objects/) — desired state, spec/status, labels, and object identity.
3. [Pods (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/workloads/pods/) and [Deployments (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/workloads/controllers/deployment/) — workload and controller behavior.
4. [Services (v1.35)](https://v1-35.docs.kubernetes.io/docs/concepts/services-networking/service/) — selectors, ports, and EndpointSlices.

## Keep open during practice

- [`kubectl` quick reference (v1.35)](https://v1-35.docs.kubernetes.io/docs/reference/kubectl/quick-reference/) — command retrieval, not passive memorization.
- [Debug Pods (v1.35)](https://v1-35.docs.kubernetes.io/docs/tasks/debug/debug-application/debug-pods/) — evidence order for broken workloads.
